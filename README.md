# LinkedInClone
A Clone project of real world LinkedIn App with core services

## LinkedIn Clone – Feature Branch: `feature/linkedInChanges`

All the latest enhancements and services for this LinkedIn clone are implemented in the **`feature/linkedInChanges`** branch. This branch contains fully functional, end-to-end features including:

- **Chat-Service** – Direct messages (DMs), conversation management, WebSocket-based real-time messaging.  
- **Connections-Service** – Sending, accepting, and managing connection requests.  
- **Posts-Service** – Creating, liking, and viewing posts.  
- **Notifications-Service** – Push notifications for connections, likes, and messages.  
- **User-Service** – User registration, authentication, and profile management.

> ✅ Everything in this branch has been tested end-to-end and is working as expected.

git checkout feature/linkedInChanges
